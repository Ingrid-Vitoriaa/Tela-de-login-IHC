# MindCare – Sistema de Login

## Descrição do Projeto
O projeto MindCare consiste em um sistema de login funcional e offline, desenvolvido para um aplicativo fictício de bem-estar mental. O sistema foi criado com foco nos princípios de Interação Humano-Computador (IHC), priorizando usabilidade, experiência do usuário e percepção de segurança.

A autenticação é simulada localmente, sem uso de banco de dados ou conexão com a internet, atendendo aos requisitos da atividade.


## Metas de Usabilidade

- **Fácil de lembrar como usar:**  
  A interface possui apenas dois campos (email e senha) e um botão de ação, facilitando a memorização do fluxo de uso.

- **Fácil de entender:**  
  Os rótulos são claros e as mensagens de feedback são diretas, permitindo que o usuário compreenda rapidamente o funcionamento do sistema.

- **Útil:**  
  O sistema cumpre sua função principal de autenticar o usuário e fornecer retorno imediato sobre a ação realizada.

- **Seguro (percepção do usuário):**  
  O campo de senha é oculto, as mensagens de erro são genéricas (não informam qual dado está incorreto) e há elementos visuais que reforçam a sensação de segurança.

- **Eficiente:**  
  O login é rápido, com validação imediata e sem etapas desnecessárias.


## Metas de Experiência do Usuário

- **Esteticamente apreciável:** uso de cores suaves e layout moderno.
- **Agradável:** interface limpa, sem excesso de informações.
- **Emocionalmente adequada:** linguagem acolhedora e visual tranquilo.
- **Satisfatória:** feedback positivo ao realizar o login com sucesso.
- **Motivadora:**  o design acolhedor incentiva o usuário a continuar utilizando a aplicação.
- **Interessante:** identidade visual alinhada ao contexto de bem-estar.
