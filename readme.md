# MindCare – Sistema de Login

## Descrição do Projeto
Este projeto consiste em um sistema de login funcional e offline para um aplicativo fictício de bem-estar mental chamado MindCare. O objetivo é aplicar conceitos de Interação Humano-Computador, com foco em usabilidade e experiência do usuário.

## Metas de Usabilidade
- **Fácil de lembrar:** Interface simples e com poucos campos.
- **Fácil de entender:** Rótulos claros e mensagens diretas.
- **Útil:** Permite autenticação do usuário.
- **Seguro (percepção):** Campo de senha oculto e mensagens genéricas de erro.
- **Eficiente:** Login rápido com feedback imediato.

## Metas de Experiência
- Esteticamente apreciável
- Agradável
- Emocionalmente adequada
- Satisfatória
- Motivadora
- Interessante

## Tecnologias Utilizadas
- HTML
- CSS
- JavaScript

## Instruções de Execução
1. Abrir o arquivo `index.html` em qualquer navegador.
2. Utilizar as credenciais:
   - Email: usuario@mindcare.com
   - Senha: 123456
3. O sistema funciona totalmente offline.
